Все самые свежие новости об лаунчере

Добавлена версия 1.19

1.19 (или же "Экваториальное обновление")" - крупное обновление Minecraft: Java Edition,
которое выйдет 26 июня 2022 года. Это обновление привнесло в игру экваториальные
леса, обновлённую пустыню и джунгли, а также большое количество новых полезных фич.

Добавлена версия 1.18.2

1.18.2 — это небольшое обновление для Java Edition, добавляющее команду /placefeature ,
изменяет систему тэгов и исправляет баги.

Добавлена версия 1.17.1

1.17.1 — это небольшое обновление для Java Edition, выпущенное 6 июля 2021 года, в
котором были изменены некоторые игровые механики и исправлен ряд ошибок.

Добавлена версия 1.16.5

Майнкрафт 1.16 анонсировали давненько - ещё на MINECON Live 2019. Обновление
посвящено нижнему миру и называется Nether update (незер апдейт) оно вышло
23.06.2020.
